The VayaPin Connector plugin enables WooCommerce store owners to offer VayaPin as a checkout helper during the checkout process.

== Description ==

This is the Wordpress & WooCommerce VayaPin plugin for enabling checkout with VayaPin in the checkout process.

The VayaPin Connector plugin allows WooCommerce store owners to offer VayaPin as a checkout helper during the checkout process. VayaPin provides a secure and convenient checkout process for customers.

== External Service Use ==

This plugin relies on the VayaPin API service to generate and retrieve VayaPin labels for orders. The API is used to fetch precise address information based on the VayaPin code.

**Service Details:**

- **Service Name:** VayaPin API
- **Service URL:** [https://cs.vayapin.com](https://cs.vayapin.com)
- **Terms of Use:** [VayaPin Terms of Use](https://vayapin.com/terms-of-use)
- **Privacy Policy:** [VayaPin Privacy Policy](https://vayapin.com/privacy-policy)

Data sent to VayaPin includes order information necessary to generate and retrieve VayaPin labels. Users must have a valid VayaPin account to use this service.

== Installation ==

1. Install the VayaPin Connector plugin either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings through the 'VayaPin' menu in the WordPress admin.
4. Enter your API credentials provided by Vayapin and save the settings.

== Frequently Asked Questions ==

= What is VayaPin? =
VayaPin is the most precise address in the world. It converts any GPS coordinate into a FREE, short and unique address, that is easy to communicate, ultimately ensuring that deliveries will reach the right place the first time, every time.

= How do I obtain VayaPin API credentials? =
You can obtain your VayaPin API credentials by signing up on the VayaPin website and creating an account.

= Does this plugin work with all versions of WooCommerce? =
This plugin is tested with WooCommerce version 3.0 and higher.
